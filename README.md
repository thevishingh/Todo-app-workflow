# 📝 Todo App Workflow

This repository is created to learn and practice **GitHub Actions ⚙️** and **CI/CD pipelines 🔄** using a simple Todo App project.

---

## 🎯 Purpose

The main goal of this repository is to understand how **workflow automation** works in GitHub.

Topics covered:

- ⚙️ GitHub Actions  
- 📄 Workflow creation  
- 🧩 Jobs and steps  
- 🔁 Running automated tasks on push or pull requests  
- 🚀 Basic CI/CD pipeline concepts  

---

## 🛠 Tech Stack

- 🟨 JavaScript  
- ⚛️ React (Todo App)  
- 🐙 GitHub Actions  

---

## 📚 Learning Goal

To understand how **CI/CD pipelines 🔄** work and how **GitHub Actions ⚙️** can automate tasks like **building, testing, and running workflows** in a repository.
